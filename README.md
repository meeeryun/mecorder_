# mecorder_
My simple video recorder using OpenCV :)


Hello, this is meeeryun :)
This Python application called MECORDER provides a simple and interactive webcam interface with multiple features for video recording, image capture, and live feed manipulation. You can control the webcam feed in real-time using keyboard shortcuts. Let me explain the manual for MECORDER.

1. RECORD – Press the Spacebar to start or stop video recording. A red REC indicator appears on the top-left corner while recording. (It will save as 'output.avi')
2. FLIP – Press the F key to toggle horizontal mirroring of the webcam feed.
3. CLICK CAPTURE – Just CLICK the screen to take a snapshot of the current frame and save it as an image file. (It will save as 'capture.png')
4. EXIT – Press ESC to safely close the webcam window and stop the program, ensuring any ongoing recording is properly saved. (It will safely close your webcam too.)

When you open the screen, you’ll see the manual on the bottom-left corner.

Thank you for reading and enjoying my MECORDER :)
